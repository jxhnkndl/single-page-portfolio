## Application Description
A clean, simple, and responsive development portfolio with a touch of unique style and personality. The deployed layout showcases a rotating array of individual and team projects, a more extend about section including tech skills badges, a prominently positioned button linking to an updated resum√©, smooth scrolling, keyframe CSS animations, detailed contact information, and a 100% unique animated avatar.
