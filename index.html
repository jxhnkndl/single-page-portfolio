Foo Bar Finder

HTML / CSS / Bootstrap / JavaScript / jQuery

The Foo Bar Finder is a double-sided application that allows a user to search for and save new recipes and cocktails by keyword or category. The application features alternate color schemes for the recipe side and drink side.

Workday Scheduler

HTML / CSS / JavaScript / jQuery

The Workday Scheduler is a dynamic planner application allowing users to input and save events spanning from 9 AM to 5 PM. Timeblocks update their background colors based on the current time.

Weather Dashboard

HTML / CSS / Bootstrap / JavaScript / jQuery

The Weather Dashboard allows users to input a city and retrieve both the city's current weather and five-day forecast. The application also allows users to save previously searched cities to later retrieve updated weather data quickly and easily.
